# Circuit Board Circuit/Circuit<sup>2</sup>

This VR racing game will allow the user pilot a car and race class mates around a track. Every few laps, all users will be asked a question on their steering wheel. If they answer it correctly, their speed will increase. 

>Note: The name is still under Review

## Questions
The questions will be asked on a interface on the wheel. Each question will have its own button the user can select with a VR pointer. 

The questions will focus on algorithm speed and big O questions. This is meant to introduce and lock in the speeds relative of each other. In order to keep students from just memorizing the correct answer certain questions can have a list of answer choices with ranking values associated with their speed. When the question is chosen a random set can be selected and the one with the highest speed ranking is the correct answer. 

#### Question Examples
1. What search algorithm is faster
    - Bubble sort
    - Selection Sort
    - Merge Sort
    - Quick Sort

2. Which Run time is faster? 
    - O(N)
    - O(N<sup>2</sup>)
    - O(Log(N))
    - O(2N)

## Driving Mechanics 

The Game will be a typical kart racing game. The vehicle should have semi realistic physics. It will be driven by the user using the VR controllers to grab a steering wheel. 

>Note: Need to figure out how to control thrust in VR.

The VR player would be located inside the car able to control thrust and steering wheels.

## Theme

The final version of the game should have the player take place in a computer themed environment. It will look as if the player is racing on a circuit/motherBoard. 
- Green, Black, Gold colors for the environment and cars
- Darker ambient lighting with Luminous objects scattered around 
- Digital Sounds

## Stretch Ideas
These are ideas that would be fun to ad but are not a priority and wont be for some time. 
#### Mario Kart like Items

Players would be able to collect items to help themselves or hinder other racers. The items could be used by throwing them or applying them using VR controls.
1. OverClock: Allows the player to get a massive temporary speed boost.

2. 404: Player creates a fake item box that does nothing

3. Miracle Sort: Player has a 1% change to double their speed

4. BlackHat: Throw a black hat at the next player, making their base speed 0 for a short time
